# Prepress Workflow Summary

1. Received customer-provided PDF
2. Review file specifications and customer notes
3. Identify prepress issues via Adobe InDesign/Creative Suite
4. Correct RGB colors, bleed, embedded fonts, and image resolution
5. Export print-ready PDF using PDF/X-1a
6. Conduct soft proof review in Adobe Acrobat
7. Prepare file for CTP (Computer-to-Plate) and production workflow
