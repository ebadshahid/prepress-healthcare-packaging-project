# Prepress Workflow Summary

1. Received customer-provided PDF
2. Reviewed file specifications and customer notes
3. Identified prepress issues via InDesign preflight
4. Corrected RGB colors, added bleed, embedded fonts, fixed image resolution
5. Exported print-ready PDF using PDF/X-1a
6. Conducted soft proof review in Acrobat
7. Prepared file for CTP and production workflow
8. Documented awareness of Illustrator & Photoshop for image/vector edits if needed
